# Sistema de Gerenciamento de Usuários (CLI)

## 📌 Descrição
Aplicação em linha de comando para gerenciamento de usuários com operações completas de CRUD. Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de backend, como persistência de dados, integração com banco de dados e organização de código.

##Funcionalidades:
- Cadastrar usuários
- Listar usuários
- Atualizar usuários
- Deletar usuários

## Tecnologias Utilizadas:
- Python
- SQLite

## Estrutura do Projeto:
- `main.py` → Responsável pela interação com o usuário e fluxo da aplicação
- `database.py` → Responsável pela conexão com o banco de dados e execução das queries

## Como Executar

1. Clonar o repositório: git clone
2. Acessar a pasta do projeto: cd user-management-cli
3. Executar a aplicação: python main.py


## Aprendizados
- Desenvolvimento de um sistema CRUD do zero
- Integração com banco de dados utilizando SQLite
- Separação de responsabilidades no código
- Uso de Git e GitHub para versionamento
